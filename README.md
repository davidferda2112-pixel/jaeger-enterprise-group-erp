# Jaeger Enterprise Group ERP

ERP web independiente para Jaeger Enterprise Group.

## Objetivo

- Frontend en Vercel sin pantalla ni aviso de Google Apps Script.
- Base operativa inicial con productos, proveedores, productos por proveedor, ordenes de compra, recepcion e inventario.
- Preparado para conectar Google Sheets / Apps Script como backend.

## Desarrollo local

```bash
npm install
npm run dev
```

## Variables

Copiar `.env.example` a `.env.local` y ajustar los valores si cambia el deployment de Apps Script.
